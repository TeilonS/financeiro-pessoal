# Controle Financeiro Pessoal — Visão Geral

## O que é
API REST em Java/Spring Boot para controle de receitas e despesas pessoais.
Categorias, saldo mensal, relatório por período, importação de extrato bancário e controle de investimentos.

## Dois objetivos
- **Uso real pessoal** — Teilon usa no dia a dia para controlar finanças
- **Portfólio** — demonstra Spring Boot 3, JWT, JPA, integração com IA

## Diferenciais
- Importação de extrato OFX e CSV (Nubank, Itaú, Inter, C6)
- Categorização automática via IA — baixa confiança fica como pendente
- Alertas de meta estourada por categoria
- Controle de investimentos com histórico mensal de evolução
- Isolamento total por usuário — cada um vê só seus dados
